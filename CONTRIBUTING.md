## Tasks
| Name        | Description                                                     |
|-------------|-----------------------------------------------------------------|
| `build`     | Generates production-grade build artefacts.                     |
| `check`     | Runs `check_fmt` and `check_ts`.                                |
| `check_fmt` | Verifies that the source code is well-formatted.                |
| `check_ts`  | Verifies that the source code is type-safe.                     |
| `fmt`       | Reformats the source code with Biome.                           |
| `install`   | Installs third-party packages and enables the Git hooks.        |
| `test`      | Runs the entire unit test suite once.                           |
| `vitest`    | Starts the Vitest UI test explorer for continuous unit testing. |
| `yolo`      | Disables the Git hooks.                                         |
